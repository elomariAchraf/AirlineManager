# AirlineManager
web app that manage an airport.
Directe Application for CRUD functions for Passengers, Pilots, Flights, and Airplanes
Used technologies Servlet, JSP, EJB, JPA, EclipseLink, Derby,JAX-RS, jersey, Glassfish, Eclipse.
